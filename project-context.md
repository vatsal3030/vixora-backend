# YouTube Clone Backend – Project Context

## Overview
This project is a full-featured YouTube-like backend built using modern backend technologies.
It supports authentication, video management, user interaction, and creator features.

The backend is already complete, tested using Postman, and production-ready.
The goal now is to build a professional frontend that consumes this API.

---

## Tech Stack

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Authentication:** JWT (Access + Refresh tokens)
- **File Uploads:** Cloudinary
- **Validation:** Zod
- **Security:** Helmet, CORS, Rate Limiting

### Frontend (to be built)
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** Context API
- **Routing:** React Router v6
- **HTTP Client:** Axios

Build a modern, responsive, scalable frontend similar to YouTube that connects seamlessly with this backend.
